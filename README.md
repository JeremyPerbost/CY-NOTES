# CY-NOTES
CY-NOTES est un programme tournant sous Windows permettant aux étudiants (notamment ceux de CY Tech ou d'autres écoles d'ingénieurs) de calculer leurs notes, leurs moyennes et de prédire les notes manquantes pour la validation de leur semestre.
Lancer simplement l'executable CY-NOTES.exe. Vous pouvez rajouter autant d'UE et de matieres que vous voulez.
Pour ajouter vos notes, veuillez modifier le fichier notes.txt et le remplir comme indiquer par le programme. Voila un exemple : 
UE INFO
5:JEE=10
3:DESIGN PATTERN=8.5
3:ARCHITECTURE RESEAU=?
2:TEST ET VERIFICATION=?
2:CYBER=5
UE MATH
4.5:STATS=15.28
4.5:IA=?
