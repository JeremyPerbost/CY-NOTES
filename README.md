# CY-NOTES
CY-NOTES est un programme tournant sous Windows permettant aux étudiants (notamment ceux de CY Tech ou d'autres écoles d'ingénieurs) de calculer leurs notes, leurs moyennes et de prédire les notes manquantes pour la validation de leur semestre.
Lancer simplement l'executable CY-NOTES.exe
